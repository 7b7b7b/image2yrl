# Image API Client 杨苒琳专用 使用教程

## 运行

双击运行：

```text
ImageApiClient.exe
```

程序会自动启动本地服务，并打开浏览器页面。无需安装 Python，也无需手动配置 API。

如果 Windows 提示“未知发布者”或安全提醒，选择“更多信息”，再选择“仍要运行”。

## 文生图

1. 在 `Prompt` 输入框里输入图片描述。
2. 选择 `Size`。
3. 选择 `Quality`，默认是 `high`。
4. `Count` 表示一次生成几张，最多 4 张。
5. 点击 `Generate`。

生成过程中，右侧会显示半透明遮罩和旋转加载提示。生成完成后，图片会显示在右侧。

## 图生图

1. 在 `Prompt` 输入框里输入你希望如何改图或参考生成。
2. 在 `Reference Images` 里选择一张或多张参考图。
3. 确认 `Size` 和 `Quality`。
4. 点击 `Generate`。

有参考图时，程序会自动使用图生图接口；没有参考图时，会使用普通文生图接口。

参考图缩略图右上角的 `x` 可以移除该参考图。

## 历史图片

底部横排会显示往期生成的图片缩略图。

点击历史缩略图后：

- 右侧会加载这张历史图片
- `Prompt` 会自动回填当时使用的提示词
- `Model`、`Size`、`Quality` 也会回填

只有使用新版程序生成的图片，才能完整回填 prompt。旧图片如果没有历史记录，只能显示图片，不能恢复 prompt。

## 图片保存位置

图片会自动保存到 exe 同目录下的 `outputs` 文件夹：

```text
ImageApiClient.exe
outputs/
```

历史记录保存在：

```text
outputs/history.json
```

不要删除 `history.json`，否则历史图片的 prompt 记录会丢失。

## 参数说明

### Size

常用选项：

```text
auto
1024x1024
1536x1024
1024x1536
2048x2048
2048x1152
3840x2160
2160x3840
```

`auto` 表示让模型自动选择合适尺寸。

### Quality

可选：

```text
high
medium
low
auto
```

`high` 质量更好，但通常会更慢、成本更高。

### Extra

高级参数，一般可以留空。

如果需要传额外参数，必须填写 JSON，例如：

```json
{"background":"transparent"}
```

## 关闭程序

关闭浏览器页面不一定会关闭后台程序。

如果需要完全退出，可以在任务管理器里结束 `ImageApiClient.exe`。
